GAMING ARENA

Project Description

This project aims at making a one time stop for old time gamers. The gaming arena will consist of 8 games which utilizes the basic concepts of computer graphics such as rendering, clipping etc. The project will consist of an initial welcome page where users can select the game he/she wants to play, the game which is chosen will hence start. Each of the member of this project is responsible for developing an individual game and for the integration of the same.
